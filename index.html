This dashboard runs itself. Fixtures, scores, league tables, last-5 form, "in action this week," the active-tournament banner, transfer & club news, the USMNT newswire, and emerging-player headlines all pull live every load — from ESPN (data) and the /api/news function (headlines). Nothing to update by hand. The only baseline lists in the code are the senior-pool roster and the prospect cards (POOL / PIPELINE), which seed the views; live news layers on top of them. To track another club, add it to CLUBS and CLUB_ESPN. Commit through the GitHub web UI; Vercel redeploys automatically.
